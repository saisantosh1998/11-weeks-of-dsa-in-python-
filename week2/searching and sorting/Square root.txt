Question:
	Given an integer x, find the square root of x.

Solution:

def floorSqrt(x): 
    if (x==0) or (x==1):
        return x
    l=1
    r=x
    while (l<=r):
        mid=(l+r)//2
        if (mid*mid==x):
            return mid
        if (mid*mid<x):
            l=mid+1
            ans=mid
        else :
            r=mid-1
    return ans
